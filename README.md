# 🚗 Car Racing Game

> 🏁 A simple and interactive car racing game built using JavaScript.

---

## 📌 Description
This project is a browser-based car racing game developed using HTML, CSS, and JavaScript. Players can control a car, avoid obstacles, and experience basic racing gameplay with interactive controls and smooth movement.

---

## 🚀 Features
- 🚗 Car movement controls  
- 🛣️ Endless racing track  
- 💥 Collision detection  
- 🎮 Interactive gameplay  
- ⚡ Lightweight and fast  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---


## ⭐ Project Purpose
This project was created to demonstrate basic game development concepts such as animation, event handling, and collision detection using JavaScript.
